CSS position values:
Relative, Absolute, and Fixed


‚Ä¢ What is the position property value?

The position property value is a positioning method for element layout. There are 5 different types of position values:

  • Static
  • Relative
  • Absolute
  • Fixed
  • Sticky

In this article, we mainly discuss the differences between relative, absolute, and fixed positioning.

‚Ä¢ Relative

An element with Position: Relative; is placed in relation to its default position. The top, right, bottom, and left properties can be used to adjust the element away from its default position. When an element is relatively positioned, its default space will remain and will not be automatically taken by other elements.

An element with Position: Relative; is often used as a parent (i.e. to nest an element inside) to a Position: Absolute; element. Why? We are talking about it just now.

‚Ä¢ Absolute

When an element is set Position: Absolute;, it will be positioned relative to the closest positioned parent. If an element with absolute position has no positioned parent, it goes up to the grandparent element or the document body.

Also, Position: Absolute; removes the positioned element from the document flow. As the example shows, when box2 is set Position: Absolute;, box3 automatically takes its default space. Box2 is now floating above box3 and can cover other elements like box1.

‚Ä¢ Fixed

You can use Position: fixed; to position an element relative to the viewport. The fixed positioned element stays in the exact same place of your screen window even if you are scrolling. So, if you want to put ads on your website, make sure you use the fixed position!
